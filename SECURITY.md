# Security Policy

## Supported versions

The latest `main` branch is supported for security fixes.

## Reporting a vulnerability

- Email: `security@privacyops.africa` (replace in your deployment if needed)
- Include: impact, reproduction steps, affected endpoints/modules, and suggested fix.
- Do not disclose publicly until maintainers confirm remediation timeline.

## Response targets

- Initial acknowledgement: within 3 business days.
- Triage decision: within 7 business days.
- Patch timeline: based on severity and exploitability.

## Scope reminders

- This project handles sensitive compliance artifacts and evidence files.
- Vulnerabilities affecting tenant isolation, auth, file upload, token handling, or audit integrity are high priority.
