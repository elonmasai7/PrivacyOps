# Production Checklist

- [ ] Strong `JWT_SECRET` configured
- [ ] PostgreSQL backups enabled
- [ ] Object storage for uploads and exports configured
- [ ] CORS restricted to production frontend domain
- [ ] TLS termination enforced
- [ ] Monitoring and alerting configured
- [ ] Sentry or equivalent error tracking configured
- [ ] Stripe keys configured (if billing enabled)
- [ ] GitHub app/token policy documented for least privilege
- [ ] Legal pages reviewed and approved by counsel
- [ ] Data retention and deletion workflows validated
