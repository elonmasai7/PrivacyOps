# Governance

## Project model

PrivacyOps Africa Core is a maintainer-led open-source project with community review lanes for framework packs, integrations, security checks, and documentation.

## Roles

- Maintainers: merge PRs, own release process, enforce quality and security policy.
- Reviewers: domain reviewers for compliance, security, and architecture.
- Contributors: submit code, docs, tests, framework packs, and integrations.

## Decision process

- Small changes: maintainer review and merge.
- Breaking changes: design proposal in issue/PR with migration notes.
- Security-sensitive changes: private review path as needed.

## Framework pack trust model

- `unverified`: community-contributed, not reviewed for production trust claims.
- `community-reviewed`: reviewed by maintainers/community specialists.
- `expert-reviewed`: reviewed by qualified privacy/security practitioners.

No pack is "official" without explicit review status and source references.
