# Roadmap

## Current release focus (v0.2)

- Harden core multi-tenant workflows and auditability.
- Expand framework pack lifecycle (import/export/review traceability).
- Ship admin controls for system settings, jobs, and health views.
- Keep AI optional and disabled by default.

## Next milestones

1. Add richer Kenya DPA/GDPR control libraries with community and expert review lanes.
2. Expand integrations (GitLab, Google Workspace, Microsoft Graph, Cloudflare, Sentry, Jira/Linear).
3. Add reminder workers and delivery channels (email, Slack, webhook retries).
4. Add stronger session lifecycle controls (refresh token rotation and revocation).
5. Add production observability stack and SLO runbooks.

## Long-term goals

- Framework pack signing and provenance verification.
- Enhanced trust center access request workflow.
- Translation/localization packs for African regions.
